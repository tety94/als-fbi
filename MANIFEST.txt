FBI Recalibration in ALS — Reproducibility Package
===================================================

This package reproduces all numbers and figures in:

  Chiò A et al. Recalibration of the Frontal Behavioural Inventory cut-off
  for behavioural screening in amyotrophic lateral sclerosis. Submitted to
  Amyotrophic Lateral Sclerosis and Frontotemporal Degeneration.

Contents
--------

How to reproduce
----------------
$ pip install -r requirements.txt
$ cd scripts
$ python 02_main_recalibration.py         # ~30 sec, replicates headline numbers
$ python 03_subscales.py                  # ~10 sec
$ python 04_sensitivity_bulbar_spinal.py  # ~10 sec
$ python 05_figures.py                    # ~15 sec

The script 01_build_dataset.py is the only one that needs the original SPSS
file (PII-bearing). The de-identified analytic dataset in data/ allows
scripts 02–05 to run as-is.

Random seed = 2026 everywhere. Bootstrap samples = 2000.
